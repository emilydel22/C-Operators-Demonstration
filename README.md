## Overview
This project demonstrates the use of **14 fundamental C operators** through a console-based program written for **EEL 2880 – Embedded Systems**.  
The program prompts the user for two integer inputs and evaluates arithmetic, relational, logical, increment, and bitwise operators, displaying results alongside the current date and time.

## Features
- User input handling via `scanf`
- Arithmetic operators (`+`, `-`, `*`, `%`)
- Relational operators (`==`, `>`, `<=`)
- Logical operators (`&&`)
- Increment operators (pre- and post-increment)
- Bitwise operators (`&`, `|`, `<<`, `>>`)
- Date and time output using `<time.h>`
